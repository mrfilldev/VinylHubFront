# VinylHub — фронтенд

SPA на React (Vite + TypeScript), клиент для VinylHub API.

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```

## API

Base URL: `https://mrfilldev-vinylhubback-5532.twc1.net`  
JWT сохраняется в `localStorage` после входа/регистрации.
